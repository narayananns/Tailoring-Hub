📊 API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PUBLIC ENDPOINTS:
─────────────────
1. POST /api/admin-management/request
   └─ Submit admin access request
      Required: name, email, password, adminCode
      Optional: reason, department, role
      Returns: Request confirmation


SUPER-ADMIN ENDPOINTS:
──────────────────────
2. GET /api/admin-management/requests/pending
   └─ View all pending requests
      Auth: Super-admin JWT token
      Returns: List of pending requests

3. GET /api/admin-management/requests/all
   └─ View complete request history
      Auth: Super-admin JWT token
      Returns: Grouped by status (pending/approved/rejected)

4. GET /api/admin-management/request/:id
   └─ View specific request details
      Auth: Super-admin JWT token
      Returns: Full request information

5. POST /api/admin-management/approve-request/:id
   └─ Approve admin request & create account
      Auth: Super-admin JWT token
      Optional: Custom permissions array
      Returns: New admin account info

6. POST /api/admin-management/reject-request/:id
   └─ Reject admin request with reason
      Auth: Super-admin JWT token
      Required: Rejection reason
      Returns: Rejection confirmation

7. DELETE /api/admin-management/cancel-request/:id
   └─ Cancel pending request
      Auth: Super-admin JWT token
      Returns: Cancellation confirmation

  _id: ObjectId,

  • password: String (hashed, required)
  • adminCode: String (hashed, required)
  
  STATUS:
  • status: String (pending|approved|rejected)
  
  APPROVAL DATA:
  • approvedBy: ObjectId (ref to Admin)
  • approvedAt: Date
  • rejectionReason: String
  • rejectedAt: Date
  
  ROLE & PERMISSIONS:
  • role: String (admin|super-admin)
  • permissions: [String] (8 possible permissions)
  
  REQUEST DETAILS:
  • reason: String
  • department: String
  • requestedAt: Date (auto)
  • updatedAt: Date (auto)
}

INDEXES RECOMMENDED:
─────────────────────
• email: unique
• status: indexed
• requestedAt: descending
• approvedBy: indexed
